#minecraft-status

This branch serve only a simple message board!

###Install
  Install node.js, and
> $ git clone https://github.com/cs8425/minecraft-status.git
> $ cd minecraft-status/
> $ git checkout message-board
> $ npm install

###Config

edit `app.js`:
```
var config = {
	port: 8080  // your port here!
};
```

###Run
> node app.js
